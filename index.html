2. Navegando na Rede Global

Voltar

Para entender como a ocorre a navegação na internet, é necessário compreender como funciona a relação entre clientes e servidores e quais os papéis de cada um.

2.1 Clientes na internet

Clientes na internet são os dispositivos ou softwares que acessam os serviços e recursos disponíveis na rede mundial. Em termos simples, um cliente é qualquer coisa que solicita e recebe informações de um servidor. Os navegadores web, como o Google Chrome ou Mozilla Firefox, são exemplos comuns de clientes. Eles permitem que os usuários visualizem páginas da web, baixem conteúdo e interajam com diferentes serviços online.

2.2 Servidores na Internet: Arquitetos da Conectividade

Voltar

Os servidores são a espinha dorsal da internet, responsáveis por armazenar, processar e fornecer dados aos clientes. Eles atendem a solicitações provenientes dos clientes, seja para exibir uma página da web, enviar um e-mail ou executar outras operações. Os servidores são robustos em termos de hardware e software, projetados para lidar com grandes volumes de tráfego e executar diversas funções simultaneamente.

2.3 Tipos de Servidores: Uma Diversidade de Funções Especializadas

Voltar

Existem vários tipos de servidores, cada um desempenhando funções específicas para atender às necessidades variadas da internet:
Servidores Web: Responsáveis por hospedar sites e entregar páginas da web aos clientes. O Apache e o Nginx são exemplos populares de software de servidor web.
Servidores de E-mail: Gerenciam o envio, recebimento e armazenamento de e-mails. Exemplos incluem o Microsoft Exchange e o Postfix.
Servidores de Banco de Dados:Armazenam e gerenciam dados. MySQL, PostgreSQL e Oracle são exemplos conhecidos nessa categoria.
Servidores de Arquivos: Facilitam o armazenamento e compartilhamento de arquivos. Servidores FTP (File Transfer Protocol) e servidores de arquivos em nuvem, como o Dropbox, são exemplos.
Servidores de Aplicativos: Executam aplicativos específicos, como servidores de jogos, servidores de mensagens instantâneas, entre outros.
Servidores de Nomes (DNS): Traduzem nomes de domínio em endereços IP, permitindo a localização de recursos na internet.
Servidores Proxy: Intermediam as comunicações entre clientes e servidores, melhorando desempenho, segurança ou filtrando conteúdo.
Essa diversidade de servidores é essencial para o funcionamento eficiente e seguro da internet, proporcionando a infraestrutura necessária para a comunicação, o armazenamento de dados e a disponibilização de serviços online.

3. Linguagem de programação

Voltar

HTML (Hypertext Markup Language): Função: Define a estrutura e o conteúdo básico de uma página web. Características: Utiliza tags para marcar elementos como títulos, parágrafos, imagens e links.

CSS (Cascading Style Sheets): Função: Responsável pela apresentação visual das páginas web.
Características: Permite a estilização de elementos HTML, controlando layout, cores, fontes e outros aspectos visuais.
JavaScript: Função: Adiciona interatividade e dinamismo às páginas web. Características: Linguagem de programação do lado do cliente que permite manipular o conteúdo da página, responder a eventos e se comunicar com servidores.
PHP(Hypertext Preprocessor): Função: Linguagem de servidor para o desenvolvimento de aplicações web dinâmicas. Características: Integrado com HTML, é utilizado para processar formulários, gerar conteúdo dinâmico e interagir com bancos de dados.

Python: Função: Versátil, é utilizado tanto no desenvolvimento web quanto em diversas outras áreas. Características: Frameworks como Django e Flask facilitam a criação de aplicativos web robustos e eficientes.

Ruby: Função: Destacado pela simplicidade e elegância, é usado principalmente no framework Ruby on Rails. Características: Favorece a convenção sobre configuração, acelerando o desenvolvimento.

Java: Função: Amplamente utilizado para aplicações web empresariais e grandes sistemas. Características: Plataforma independente e orientada a objetos, com frameworks como Spring MVC.

TypeScript: Função: Extensão do JavaScript que adiciona tipagem estática, facilitando o desenvolvimento e manutenção de grandes projetos. Características: Compatível com JavaScript e amplamente utilizado em projetos Angular.
Go (Golang): Função: Desenvolvimento eficiente de servidores web e aplicações concorrentes. Características: Criada pela Google, destaca-se pela performance e simplicidade.

Swift: Função: Principalmente associada ao desenvolvimento de aplicativos para dispositivos Apple. Características: Rápida e segura, é a linguagem de programação preferencial para iOS e macOS. A escolha da linguagem de programação depende das necessidades específicas do projeto, das preferências da equipe de desenvolvimento e das características desejadas na aplicação web. Cada linguagem tem seus pontos fortes e é adequada para diferentes contextos, contribuindo para a diversidade e flexibilidade do desenvolvimento web.
